Lilaq [/ˈlaɪlək/] is a powerful plotting library for [Typst](https://typst.app). 

- Publication-ready graphics. 
- Real-time preview. 
- Seamless integration with Typst documents. 
- Consistent styling and interoperability with [Zero](https://typst.app/universe/package/zero). 
- Powerful configuration abilities.
